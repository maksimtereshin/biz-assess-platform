# Дорожная карта продукта

## MVP — Основной функционал (этап 1)

1. [x] **Базовый Telegram-бот и админка**
   Настроить приветствие, обработку команд, логирование и базовую панель администратора для аналитики (кто, что и как сделал в системе - возможно просто выгрузка в X).
   `M`

2. [ ] **Мини-приложение Telegram (Mini App)**
   Реализовать интерфейс опроса с пошаговым прохождением, кнопками, оплатой и передачей результата в бот.
   `L`

3. [ ] **Механика опросов (full/express)**
   Реализовать 2 типа опросов, включая сохранение состояния и возобновление с места остановки:
   - full: 7 категорий, 96 вопросов
   - express: 6 категорий, 60 вопросов
   `L`

4. [x] **Результаты**
   Просмотр пройденных отчетов только через телеграм бот, вкладка Мои результаты. Каждый отчет должен быть доступен для скачивания в формате PDF.
   `M`

5. [ ] **Платёжный модуль**
   Интегрировать Telegram Payments / Stripe / YooKassa для покупки:
   - повторного прохождения опроса (не разрешено проходить отчет повторно без оплаты)
   - детализации по категориям
   `M`

7. [ ] **Реферальная система**
   Реализовать персональные реферальные ссылки с отслеживанием переходов и начислением бонусов.
   `M`
