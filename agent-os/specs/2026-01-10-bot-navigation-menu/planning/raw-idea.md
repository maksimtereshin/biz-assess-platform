# Raw Idea: Стартовый экран бота с навигационным меню

**Date:** 2026-01-10

## Описание фичи

**Стартовый экран бота с навигационным меню**

### Функционал:
- При открытии бота показывается приветственное сообщение и кнопка старт (чтобы не вводить руками /start)
- Приветственное сообщение - краткое сообщение с именем юзера

### Главное меню (6 кнопок):
1. Чекап - переход в подменю Чекапа
2. Запись на старт сессию (предупреждение - страница в разработке)
3. О проекте (предупреждение - страница в разработке)
4. FAQ (предупреждение - страница в разработке)
5. Админ панель (доступна только для Админов)

Кнопка меню рядом с поисковой строкой имеет все те же кнопки, при нажатии открывается выбранное меню и дальше взаимодействие пользователя с системой стандартное.

### Подменю "Чекап" (3 кнопки):
1. Начать Чекап
2. Мои результаты
3. Реферальная программа (предупреждение - страница в разработке)

### Навигация:
Во всех пунктах меню, кроме стартового экрана нужна кнопка Назад, чтобы перейти на уровень выше.
