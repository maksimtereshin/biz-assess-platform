# Идея: Админ-панель для управления контентом (CMS)

## Описание фичи

Полноценная система управления контентом для опросов, вопросов, ответов и категорий БЕЗ привлечения разработчика.

## Основные возможности

- **CRUD опросов**: создание и редактирование опросов (Express/Full/отраслевые)
- **Управление вопросами**: добавление вопросов с различными типами (single choice, multiple choice, scale)
- **Настройка категорий**: управление категориями и весами для расчета результатов
- **Визуальный редактор**: редактор с предпросмотром изменений

## Целевой результат

Администратор заходит в систему → конфигурирует опросы → изменения применяются автоматически в Telegram боте и WebApp без участия разработчика.

## Технологическая база

- **AdminJS** с интеграцией TypeORM
- Интеграция с существующей базой данных PostgreSQL
- Использование существующих TypeORM entities

## Приоритет

**Критический** — Phase 2, топ-1 приоритет

## Размер

**XL** — большая фича, требующая значительной разработки

## Целевая аудитория

- Владелец продукта
- Контент-менеджер
- Администратор системы

## Бизнес-ценность

- Независимость от разработчиков при создании нового контента
- Быстрое масштабирование продукта (новые опросы, новые отрасли)
- Снижение времени на внесение изменений с недель до минут
- Возможность A/B тестирования различных формулировок вопросов
